# PyGame-太空生存戰
## 開啟
將Game.zip 解壓縮後，開啟MainGame.exe

## 移動控制(move)
方向鍵(左/右) or A/D 

## 攻擊(attack)
空白鍵(space)

## 離開遊戲(quit)
直接關閉視窗
